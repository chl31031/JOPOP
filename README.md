# JOPOP (팝업스토어 통합 정보 및 예약 시스템)

JOPOP은 오프라인 팝업스토어 정보를 통합하여 사용자들이 한눈에 확인하고,  
예약 및 방문 계획을 세울 수 있도록 돕는 웹 애플리케이션입니다.

## 주요 기능
- 팝업스토어 목록 확인 및 상세 정보 보기
- 날짜 및 위치 기반 필터링 기능
- 사용자 로그인 및 회원가입
- 관리자 전용 상품 등록/수정/삭제 기능
- 게시판 커뮤니티 (팁 공유, 후기 작성 등)

## 기술 스택
- Java, JSP, Servlet
- Oracle SQL
- HTML, CSS, JavaScript
- Apache Tomcat

## 내가 담당한 부분
- 사용자 회원가입 시 일반 사용자와 관리자 구분 기능 구현
- 관리자 계정으로 상품 등록, 수정, 삭제 기능
- 상품 정보의 DB 연동 및 SQL 작성
- 일부 JSP UI 개선 및 테스트 참여

## 실행 방법
1. GitHub 저장소 클론

2. Java 웹 프로젝트 환경(Eclipse 또는 IntelliJ 등)에서 프로젝트 불러오기

3. Apache Tomcat 서버와 Oracle DB 연결 설정

4. 브라우저에서 `http://localhost:8080/JOPOP` 접속하여 실행
